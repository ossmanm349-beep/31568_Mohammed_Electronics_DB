# 31568 - Electronics Inventory and Sales Management System

## Project Overview
This repository contains the complete database project for the Electronics Inventory and Sales Management System. The project focuses on handling electronics stock control, tracking business transactions, implementing automated database constraints, and providing comprehensive reporting.

## Repository Contents
* **31568_Mohammed_Database_Screenshots.zip**: Contains categorized screenshots of tables, constraints, data insertions, triggers (Stock Before/After), and packages.
* **31568_Mohammed_Electronics_Dashboard.xlsx**: Excel dashboard for monitoring stock levels and sales data.
* **31568_Mohammed_Project_Proposal.pptx**: Presentation outlining the project's core concept, problem statements, and proposed solutions.
* **31568_Mohammed_Final_Presentation.pptx**: The final 10-slide project presentation.

## Database Features Implementation
* Relational Tables & Constraints Setup
* Sample Data Insertion
* Advanced Business Logic using Triggers (Stock Before & After Triggers)
* Procedural Logic via Database Packages and Procedures
* Complex Selection Queries for Reporting
